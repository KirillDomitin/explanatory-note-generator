# Explanatory Note Generator

Генератор пояснительной записки к бухгалтерской отчётности за 2025 год на основе данных из ЕГРЮЛ по ИНН.

Сервис принимает ИНН организации, запрашивает актуальные данные через открытый API (egrul.itsoft.ru), парсит их и заполняет готовый .docx-шаблон с помощью Jinja-подобных плейсхолдеров.

### Технологии
- Python 3.12
- FastAPI + Uvicorn
- docxtpl (Jinja-подобные шаблоны в Word)
- requests
- uv (для управления зависимостями)

### Функционал
- Получение данных по ИНН
- Парсинг основных сведений: название, адрес, дата регистрации, уставный капитал, участники, ОКВЭД, генеральный директор
- Заполнение шаблона пояснительной записки
- Скачивание готового .docx

### Требования
- Python 3.12+
- Docker (опционально)

### Установка и запуск (локально)

1. Клонируй репозиторий
   ```bash
   git clone <твой-репозиторий>
   cd explanatory_note
   

2. Установи зависимости через uv
    ```Bash
   uv sync
3. Запусти сервер
   ```Bash
   uv run uvicorn app:app --reload --port 8000

4. Открой в браузере: http://127.0.0.1:8000/docs Вызов эндпоинта (пример):
   ```text
   http://127.0.0.1:8000/generate/?inn=770101001
    ```
   скачается заполненный файл Пояснения_{inn}.docx

### Запуск в Docker

1. Собери образ
   ```Bash
    docker build -t docx-generator .
   ```

2. Запусти контейнер
    ```Bash
    docker run -d -p 8000:8000 --name docx-app docx-generator
    ```


3. Проверь логи
   ```Bash
   docker logs -f docx-app
   ```
    Доступ: http://localhost:8000/docs или по IP сервера
